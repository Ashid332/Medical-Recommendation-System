# Jupyter Notebooks

This directory contains Jupyter notebooks for exploratory data analysis, model development, and visualization.

## Notebooks

### 1. Data Exploration
- **File**: `01_data_exploration.ipynb`
- **Description**: Initial data analysis, statistical summaries, and data quality checks
- **Output**: Dataset statistics, missing value analysis, data distribution plots

### 2. Feature Engineering
- **File**: `02_feature_engineering.ipynb`
- **Description**: Feature creation, transformation, and selection
- **Output**: Processed features, feature importance rankings

### 3. Model Development
- **File**: `03_model_development.ipynb`
- **Description**: Model training, hyperparameter tuning, and cross-validation
- **Output**: Trained models, performance metrics, comparison charts

### 4. Results Analysis
- **File**: `04_results_analysis.ipynb`
- **Description**: Detailed analysis of model predictions and error patterns
- **Output**: Confusion matrices, ROC curves, feature importance plots

## Usage

To run the notebooks:

```bash
jupyter notebook
```

Then select the desired notebook from the file browser.

## Requirements

- pandas
- numpy
- scikit-learn
- matplotlib
- seaborn
- jupyter

Install with: `pip install -r requirements.txt`
